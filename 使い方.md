# Compare-ShowLogs_ByIP.ps1 使い方ガイド

`Compare-ShowLogs_ByIP.ps1` は、Cisco の `show` コマンドで取得したログを、同一 IP ごとに比較して差分レポートを出力する PowerShell スクリプトです。

もともとは、Tera Term マクロで取得した日課点検ログの確認を想定して作成しています。特に [Termnix-IT/Teratarm_macro](https://github.com/Termnix-IT/Teratarm_macro) のように、Cisco 系機器へ接続して複数の `show` コマンドを自動実行するマクロから取得したログを比較する用途に向いています。

## できること

- 対象 IP の最新ログを基準に比較する
- 特定ファイルを基準に比較する
- 約 1 日前、約 1 週間前のログを自動選定する
- ノイズ行を除外して差分を見やすくする
- 差分結果を UTF-8 の `.txt` レポートとして保存する

## 前提条件

- Windows PowerShell 5.1 以上、または PowerShell 7 以上
- ログファイルが UTF-8 で保存されていること
- Tera Term マクロや手動操作などで、比較対象の `show` コマンドログを事前に取得していること
- ログファイル名が次の形式であること

```text
YYMMDD_HHMMSS_IP_from_USER.log
```

例:

```text
260412_083000_10.0.0.1_from_admin.log
260413_083500_10.0.0.1_from_admin.log
260405_084000_10.0.0.1_from_admin.log
```

## 想定しているログ取得元

このスクリプトは、Tera Term マクロで定期取得した Cisco ログの差分確認を主な用途として想定しています。

参考:

- [Termnix-IT/Teratarm_macro](https://github.com/Termnix-IT/Teratarm_macro)

このマクロは、README 上では Cisco 系機器に対して `terminal length 0` や複数の `show` コマンドを順次実行する簡易マクロとして説明されています。本スクリプトは、そのように取得したログを日次・週次で比較確認する使い方に適しています。

## 既定の動作

引数を指定せずに実行すると、次の設定で動作します。

- ログ格納先: `$HOME\Desktop\ログ保管`
- 出力先: `<LogDirectory>\出力`
- 比較元: ログ全体の中で最も新しいファイル
- 比較先:
  - 比較元の約 1 日前に最も近い同一 IP のログ
  - 比較元の約 1 週間前に最も近い同一 IP のログ

実行例:

```powershell
.\Compare-ShowLogs_ByIP.ps1
```

## 基本的な使い方

### 1. 特定 IP の最新ログを基準にする

指定した IP の中で最も新しいログを比較元にします。

```powershell
.\Compare-ShowLogs_ByIP.ps1 -TargetIP "10.0.0.1"
```

### 2. 特定ファイルを基準にする

比較元を明示したい場合は `-CurrentFile` を使います。

```powershell
.\Compare-ShowLogs_ByIP.ps1 -CurrentFile "$env:USERPROFILE\Desktop\ログ保管\260412_083000_10.0.0.1_from_admin.log"
```

### 3. ログ保存場所を変更する

既定の `Desktop\ログ保管` 以外を使う場合は `-LogDirectory` を指定します。

```powershell
.\Compare-ShowLogs_ByIP.ps1 -LogDirectory "C:\Logs\Cisco"
```

### 4. 出力先を変更する

レポート保存先を変えたい場合は `-OutputDirectory` を指定します。

```powershell
.\Compare-ShowLogs_ByIP.ps1 -LogDirectory "C:\Logs\Cisco" -OutputDirectory "C:\Logs\Cisco\Report"
```

### 5. サブフォルダも含めて検索する

ログが複数フォルダに分かれている場合は `-Recurse` を使います。

```powershell
.\Compare-ShowLogs_ByIP.ps1 -LogDirectory "C:\Logs\Cisco" -Recurse
```

## 差分を見やすくするオプション

### IgnorePattern

`-IgnorePattern` には正規表現を複数指定できます。条件に一致した行は比較対象から除外されます。

たとえば、空行や uptime 系の変動しやすい行を除外したい場合:

```powershell
.\Compare-ShowLogs_ByIP.ps1 -TargetIP "10.0.0.1" -IgnorePattern '^\s*$', 'uptime', 'Last input'
```

注意:

- 除外した行は比較対象から外れます
- ただしレポートに出る行番号は、元ファイルの行番号のままです

## 比較対象の時間ずれを調整する

比較先は「ちょうど 1 日前」「ちょうど 1 週間前」ではなく、その時刻に最も近いログを選びます。

許容誤差は次の既定値です。

- `OneDayToleranceHours`: `36`
- `OneWeekToleranceHours`: `96`

必要に応じて変更できます。

```powershell
.\Compare-ShowLogs_ByIP.ps1 -TargetIP "10.0.0.1" -OneDayToleranceHours 24 -OneWeekToleranceHours 72
```

## 出力されるファイル

条件に合う比較対象が見つかると、出力先フォルダに次のようなファイルが生成されます。

```text
diff_1day_<base>_<timestamp>.txt
diff_1week_<base>_<timestamp>.txt
```

例:

```text
diff_1day_260413_083500_10.0.0.1_from_admin_20260414-221500.txt
diff_1week_260413_083500_10.0.0.1_from_admin_20260414-221500.txt
```

## レポートの見方

レポートには次の情報が含まれます。

- 生成日時
- 比較元ファイル
- 比較元ログ日時
- 比較先ファイル
- 比較先ログ日時
- 除外したパターン
- 差分一覧
- 総変更件数

差分は主に次の 2 種類です。

- `Added`: 現在のログに追加されている行
- `Removed`: 過去のログにしか存在しない行

## 実用的な実行例

### よく使う最小例

```powershell
.\Compare-ShowLogs_ByIP.ps1 -TargetIP "10.0.0.1"
```

### 再帰検索付きで比較

```powershell
.\Compare-ShowLogs_ByIP.ps1 -LogDirectory "C:\Logs\Cisco" -TargetIP "10.0.0.1" -Recurse
```

### ノイズ除外付きで比較

```powershell
.\Compare-ShowLogs_ByIP.ps1 `
  -LogDirectory "C:\Logs\Cisco" `
  -OutputDirectory "C:\Logs\Cisco\出力" `
  -TargetIP "10.0.0.1" `
  -IgnorePattern '^\s*$', 'uptime', 'Last input', '^Load for'
```

### 基準ファイルを固定して比較

```powershell
.\Compare-ShowLogs_ByIP.ps1 `
  -CurrentFile "C:\Logs\Cisco\260413_083500_10.0.0.1_from_admin.log" `
  -OneDayToleranceHours 24 `
  -OneWeekToleranceHours 72
```

## 戻り値

スクリプトはパイプラインに次の情報を返します。

```text
OutputPath
ChangeCount
```

PowerShell 上で受ける例:

```powershell
$result = .\Compare-ShowLogs_ByIP.ps1 -TargetIP "10.0.0.1"
$result | Format-Table
```

## エラーや警告になりやすいケース

### 命名規則に合わない

`YYMMDD_HHMMSS_IP_from_USER.log` に合わないファイルはスキップされます。

### 対象 IP のログがない

`-TargetIP` で指定した IP のログが存在しない場合はエラーになります。

### 許容時間内に比較先がない

約 1 日前、または約 1 週間前に相当するログが見つからない場合、その比較レポートは作成されません。

### 文字コードが UTF-8 ではない

文字化けや比較結果の不整合が出る可能性があります。

## 実行前チェック

構文だけ確認したい場合:

```powershell
[System.Management.Automation.Language.Parser]::ParseFile(".\Compare-ShowLogs_ByIP.ps1", [ref]$null, [ref]$null)
```

サンプル実行:

```powershell
.\Compare-ShowLogs_ByIP.ps1 -LogDirectory ".\tmp\logs" -OutputDirectory ".\tmp\out" -TargetIP "10.0.0.1"
```

## 補足

- 出力先フォルダが存在しない場合は自動で作成されます
- 出力先フォルダ直下の `.log` は入力検索から除外されます
- 差分比較は LCS ベースです
- 大きなログでも比較しやすいよう、共通の先頭・末尾は事前に除外してから比較します
